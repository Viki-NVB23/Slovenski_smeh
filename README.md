🎤 Slovenski Smeh

Mobilna aplikacija za raziskovanje slovenskega stand-up sveta 🇸🇮

📱 Opis projekta

Slovenski Smeh je mobilna aplikacija, ki na enem mestu združuje informacije o slovenskih stand-up komikih, njihovih nastopih in izvoru. Namenjena je vsem ljubiteljem stand-up komedije, ki želijo hitro in enostavno dostopati do aktualnih dogodkov in informacij o komikih.
